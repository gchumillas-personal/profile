Cómo funciona

El desarrollo de una aplicación consta de varias etapas, durante las cuales se mantendrá el contacto con el cliente para asegurarnos que sigue la dirección adecuada.

I. Recopilación de requisitos

Durante esta primera toma de contacto se inicia la conversación con el cliente mediante video conferencia o correo electrónico y se determinan los requisitos que debe cumplir la aplicación.

II. Diseño de las plantillas

Antes de iniciarnos en la etapa de desarrollo es importante determinar qué pantallas conformarán la aplicación y su interacción entre ellas. Para ello se diseñarán una serie de plantillas sujetas a la aprovación del cliente.

III. Desarrollo

Una vez el cliente ha dado su conformidad al diseño iniciaremos la fase de desarrollo. Básicamente consiste en "traer a la realidad" las distintas plantillas que hemos diseñado en la fase anterior. Cada cierto tiempo se publicarán los avances, de manera que el cliente pueda ver el progreso de su aplicación.

IV. Instalación

Una vez el desarrollo ha finalizado y la aplicación funciona correctamente, se procederá a su instalación para su uso en producción.

Cuánto cuesta

Mi tarifa actual son 45€ la hora y cada etapa de desarrollo se factura y se paga de forma independiente. Como ejemplo ilustrativo, supongamos que tenemos un blog y queremos gestionar las entradas. La aplicación deberá ser capaz de listar, borrar, editar o insertar nuevas entradas. En total serían cuatro pantallas, incluyendo la pantalla de identificación.
